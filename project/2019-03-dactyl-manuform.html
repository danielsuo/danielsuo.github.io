Dactyl-Manuform Keyboard: Where keyboard keyboards you!

A tented, ortholinear keyboard with molded thumb clusters.

Work in progress. Photo album here. Code here. A cross between a Dactyl keyboard and a Manuform keyboard. Or a cross between Kinesis Advantage and Kinesis Freestyle.

Features

  • Ortholinear (keys are in a column)
  • Hot-swappable mechanical keys
  • Molded thumb clusters
